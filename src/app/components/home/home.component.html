O que é

Ciencia de dados?

A ciência de dados é uma area de tecnologia que utiliza técnicas de análise de dados para extrair informações valiosas e insights a partir de grandes volumes de dados. Ela combina elementos de estatística, matemática, ciência da computação e domínio de negócios para resolver problemas e tomar decisões informadas.
